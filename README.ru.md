> **Zero-Trust AI Ассистент** — Безопасный, локальный ИИ с усиленной изоляцией

ClosedPaw — это AI-ассистент, ориентированный на приватность, который работает полностью на вашем локальном компьютере. В отличие от облачных решений, ваши данные никогда не покидают устройство. Построен на архитектуре security-first с использованием gVisor/Kata Containers для настоящей изоляции.

## 🚀 Быстрый старт

### Рекомендуемая платформа

**Linux или macOS настоятельно рекомендуются** для лучшего опыта безопасности:

- ✅ **Полная изоляция gVisor/Kata** — Настоящая изоляция на уровне ядра
- ✅ **Нативная контейнерная безопасность** — Без накладных расходов виртуализации
- ✅ **Лучшая производительность AI** — Прямой доступ к GPU
- ⚠️ **Ограничения Windows** — Только Docker Desktop или WSL2; полная изоляция недоступна в Windows Home

### Установка через npm (Кроссплатформенная)

```bash
npm install -g closedpaw
```

### Установка одной командой (Альтернатива)

**Linux / macOS (Рекомендуется):**
```bash
curl -sSL https://raw.githubusercontent.com/logansin/closedpaw/main/installer/install.sh | bash
```

**Windows (PowerShell):**
```powershell
iwr -useb https://raw.githubusercontent.com/logansin/closedpaw/main/installer/install.ps1 | iex
```

### Ручная установка

```bash
# Клонировать репозиторий
git clone https://github.com/logansin/closedpaw.git
cd closedpaw

# Установить бэкенд
cd backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt

# Установить фронтенд
cd ../frontend
npm install

# Запустить сервисы
npm run dev  # Запускает бэкенд и фронтенд
```

## 🔒 Функции безопасности

- **Архитектура Zero-Trust** — Нет неявного доверия, все действия проверяются
- **Усиленная изоляция** — gVisor/Kata Containers (не просто Docker)
- **Защита от prompt injection** — Защита от атак типа CVE-2026-25253
- **Только локальная работа** — Ollama на 127.0.0.1, Web UI на localhost
- **Human-in-the-Loop** — Критические действия требуют подтверждения
- **Аудит логирования** — Все действия логируются для forensic анализа
- **Шифрованное хранилище** — API ключи шифруются при хранении

## 🛡️ Реальная оценка безопасности

> **Ни одна система не является 100% безопасной.** Мы не претендуем на совершенство — мы претендуем на *максимально возможную защиту*.

### От чего мы защищаем

| Угроза | Уровень защиты | Примечания |
|--------|----------------|------------|
| Prompt Injection | ✅ Высокий | Многоуровневая защита, санитизация входных данных |
| Выполнение кода | ✅ Высокий | Песочница gVisor, фильтры seccomp |
| Утечка данных | ✅ Высокий | Только локально, шифрованное хранилище |
| Сетевые атаки | ✅ Высокий | Привязка к 127.0.0.1, нет внешнего доступа |
| Supply Chain | ⚠️ Средний | Подписанные пакеты, сканирование зависимостей |
| Физический доступ | ❌ Низкий | Рекомендуется шифрование на уровне ОС |

### Защита в глубину

ClosedPaw реализует **защиту в глубину** — множество перекрывающихся уровней безопасности:

```
┌─────────────────────────────────────────┐
│  Уровень 1: Валидация входных данных    │
│  Уровень 2: Фильтры prompt injection    │
│  Уровень 3: Изолированное выполнение    │
│  Уровень 4: Human-in-the-Loop           │
│  Уровень 5: Аудит логирование           │
│  Уровень 6: Шифрованное хранилище       │
└─────────────────────────────────────────┘
```

**Если один уровень fails, другие защитят вас.**

### Почему размер имеет значение

> **112 МБ** — это вес защиты.

```
Размер пакета:
├── 🛡️ gVisor/Kata Runtime     ~15 МБ
├── 🔐 Криптографический стек  ~25 МБ  (PyNaCl, Cryptography)
├── 🤖 AI защитные слои        ~20 МБ  (фильтры, валидаторы)
├── 📡 Каналы связи            ~15 МБ  (Telegram, Discord, Slack)
├── 🎨 Next.js Web UI          ~37 МБ
└── Итого: Защита, которой можно доверять
```

**Меньший размер = меньше защиты.** Мы не извиняемся за то, что защищаем вас правильно.

### Сравнение

| Продукт | Размер | Изоляция | HITL | Шифрование |
|---------|--------|----------|------|------------|
| "Лёгкие" AI инструменты | 5-10 МБ | ❌ Нет | ❌ Нет | ❌ Нет |
| OpenClaw | ~50 МБ | ⚠️ Только Docker | ❌ Нет | ⚠️ Частично |
| **ClosedPaw** | **112 МБ** | **✅ gVisor/Kata** | **✅ Да** | **✅ Полное** |
